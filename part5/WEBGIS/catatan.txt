helper
function assets($a=''){
    $getbase_assets=$GLOBALS['setUri']['assets'];
    return base_url($getbase_assets.$a);
}

function url($a='',$b=''){
    return base_url($b.'?halaman='.$a);
}

function redirect($a=''){
  header("location: ".$a);
  exit;
}

function templates($a=''){
    return assets($GLOBALS['template'].$a);
}

Penjelasan Elemen Kode
<li class="treeview">

Ini adalah elemen daftar (list item) dengan kelas CSS treeview. Elemen ini biasanya digunakan dalam konteks navigasi atau menu yang dapat di-expand/collapse.
<a href="#">

Ini adalah elemen tautan (anchor). Tanda # berarti tautan ini tidak mengarah ke mana pun (placeholder link). Dalam konteks navigasi, ini sering digunakan untuk elemen yang dapat diklik untuk membuka submenu.
<i class="fa fa-files-o"></i>

Ini adalah elemen ikon yang menggunakan Font Awesome (indicated by the class fa). Ikon yang digunakan adalah fa-files-o.
<span>tmt name</span>

Ini adalah elemen span yang digunakan untuk menampilkan teks "tmt name".
<span class="pull-right-container">

Ini adalah elemen span dengan kelas pull-right-container. Elemen ini biasanya digunakan untuk menampung elemen lain yang harus diposisikan di sebelah kanan elemen induknya.
<span class="label label-primary pull-right">4</span>

Ini adalah elemen span dengan kelas label label-primary pull-right. Elemen ini menampilkan label dengan teks "4" dan diposisikan di sebelah kanan.
<ul class="treeview-menu">

Ini adalah elemen daftar tak berurutan (unordered list) dengan kelas treeview-menu. Ini menandakan submenu yang akan ditampilkan saat elemen treeview di klik.
Sub-Item <li> dalam treeview-menu

Setiap elemen <li> dalam treeview-menu adalah item menu yang merupakan bagian dari submenu. Masing-masing mengandung tautan (<a>) ke berbagai halaman dengan ikon fa-circle-o dari Font Awesome.

<div class="content-wrapper">

Elemen div ini membungkus seluruh konten halaman. Kelas content-wrapper digunakan untuk mengatur tampilan wrapper utama dari halaman.
<section class="content-header">

Bagian ini merupakan header dari konten. Di sini terdapat elemen <h1> yang memberikan judul "Blank page" dengan deskripsi kecil "it all starts here".
<ol class="breadcrumb">

Ini adalah elemen breadcrumb, yang membantu navigasi pengguna dengan menunjukkan lokasi halaman saat ini dalam struktur situs. Elemen li dalam ol menunjukkan jalur navigasi:
Tautan ke "Home" dengan ikon dashboard (fa-dashboard).
Tautan ke "Examples".
Item aktif "Blank page".
<section class="content">

Bagian ini berisi konten utama dari halaman. Di dalamnya terdapat kotak (box) default.
<div class="box">

Elemen ini merupakan sebuah kotak (box) yang digunakan untuk menampilkan konten spesifik. Di dalamnya ada beberapa bagian:
Header (<div class="box-header with-border">): Berisi judul kotak ("Title") dan alat-alat (tools) seperti tombol untuk menutup atau mengecilkan kotak.
Body (<div class="box-body">): Bagian utama dari kotak yang berisi teks "Start creating your amazing application!".
Footer (<div class="box-footer">): Bagian bawah kotak yang berisi teks "Footer".
Tombol Alat (Tools)

Tombol alat di header kotak menggunakan ikon Font Awesome untuk memberikan fungsi interaktif:
Tombol dengan ikon fa-minus digunakan untuk mengecilkan (collapse) kotak.
Tombol dengan ikon fa-times digunakan untuk menghapus (remove) kotak.
